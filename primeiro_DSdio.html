Surgimento da internet

(Voltar)

A internet começou a se desenvolver nos anos 1960, quando o Departamento de Defesa dos EUA criou a ARPANET, uma rede de comunicação resistente a falhas, conectando computadores de universidades e centros de pesquisa. Nos anos 1980, a ARPANET expandiu-se e, com a padronização de protocolos como o TCP/IP, formou a base da internet. Em 1991, Tim Berners-Lee lançou a World Wide Web, facilitando o acesso à informação com páginas web e hyperlinks, o que popularizou a internet. Desde então, ela se tornou uma ferramenta essencial na vida moderna, impactando a comunicação, o trabalho e o consumo de informações.

Como funciona o HTML

(Voltar)

O HTML (HyperText Markup Language) é a linguagem usada para criar e organizar o conteúdo da web. Ele é composto por elementos, representados por tags, que estruturam a página, como cabeçalhos, parágrafos, imagens e links. As tags normalmente vêm em pares, com uma de abertura e outra de fechamento, como a tag <p> para parágrafos. Além de texto, HTML permite incorporar multimídia, como imagens e vídeos, tornando as páginas mais interativas. Embora o HTML defina a estrutura, ele é combinado com CSS, para estilo visual, e JavaScript, para interatividade, resultando em páginas mais completas e dinâmicas.

O que são Tags

(Voltar)

As tags são essenciais para a construção de documentos HTML, pois indicam aos navegadores como exibir o conteúdo. Elas podem estruturar títulos, parágrafos, listas e imagens, ajudando os desenvolvedores a organizar a informação. As tags são divididas em abertura, que inicia um elemento, como <h1> para títulos, e fechamento, que encerra o elemento, como </h1>. Existem também tags auto-fechadas, como <br> para quebra de linha e <img> para imagens, que não precisam de tags de encerramento. Além disso, tags podem ter atributos que fornecem informações extras, como o atributo <href> na tag <a>, que define o destino de um link. A combinação de tags e atributos oferece controle detalhado sobre o conteúdo na web.

Tags de Cabeçalho

(Voltar)

As tags de cabeçalho em HTML, que variam de <h1> até <h6>, são usadas para definir títulos e subtítulos em uma página web, onde <h1> é o mais importante e <h6> é o menos importante. A tag <h1> é utilizada para o título principal da página e deve ser usada apenas uma vez por página para otimização em mecanismos de busca (SEO). A tag <h2> serve para marcar as seções principais do conteúdo, enquanto <h3> é usada para subtítulos dentro dessas seções definidas por <h2>. As tags <h4>, <h5> e <h6> seguem essa hierarquia, sendo usadas para subseções dentro das seções definidas pelas tags de nível imediatamente superior, com <h6> representando o menor nível de título.

Parágrafo

(Voltar)

A tag <p> é utilizada para definir um parágrafo de texto em HTML, criando uma separação clara entre blocos de texto ao adicionar espaçamento acima e abaixo do conteúdo. Ela é essencial para organizar e formatar o texto em uma página, como no exemplo <p> Este é um parágrafo.</p> , que cria um parágrafo contendo o texto "Este é um parágrafo."

Marcação

(Voltar)

A tag <mark> é usada para destacar partes do texto, normalmente com um fundo amarelo, para indicar que essa parte é relevante ou merece atenção especial. Um exemplo seria <p> Este é um texto destacado.</p>, que destaca a expressão "texto destacado" dentro de um parágrafo.

Texto Pequeno

(Voltar)

A tag <small> é usada para exibir texto em um tamanho menor que o padrão, sendo comumente aplicada em notas de rodapé ou informações complementares. Por exemplo, a marcação <small>Texto menor que o normal.</small> renderiza o texto "Texto menor que o normal" em um tamanho reduzido.

Texto Itálico

(Voltar)

A tag <i> é utilizada para exibir texto em itálico, sendo comumente empregada para dar ênfase a palavras ou frases. Um exemplo seria a marcação <i>Texto em itálico.</i>, que renderiza o texto "Texto em itálico" com estilo inclinado.

Texto Sublinhado

(Voltar)

A tag <u> é utilizada para sublinhar texto, embora não deva ser empregada apenas para ênfase, pois o sublinhado é mais comumente associado a links. Um exemplo dessa marcação é <u>Texto sublinhado.</u>, que renderiza o texto "Texto sublinhado" com uma linha abaixo.

Texto em Negrito

(Voltar)

A tag <strong> é utilizada para enfatizar o texto, sendo normalmente renderizada em negrito. O conteúdo dentro dessa tag indica uma importância especial, como no exemplo <strong>Texto em negrito.</strong>, que exibe a frase "Texto em negrito" com destaque.

Lista Ordenada

(Voltar)

A tag <ol> é utilizada para criar uma lista ordenada, na qual os itens são numerados automaticamente. Por exemplo, a marcação <ol><li>Item 1</li><li>Item 2</li></ol> gera uma lista com os itens "Item 1" e "Item 2", exibidos em ordem numérica.

Lista Não Ordenada

(Voltar)

A tag <ul> é utilizada para criar uma lista não ordenada, onde os itens são apresentados com marcadores, como bolinhas. Um exemplo dessa marcação é <ul><li>Item A</li><li>Item B</li></ul>, que gera uma lista com os itens "Item A" e "Item B", cada um precedido por um marcador.

Item de Lista

(Voltar)

A tag <li> é utilizada para definir um item em uma lista, podendo ser parte de uma lista ordenada (<ol>) ou não ordenada (<ul>). Por exemplo, a marcação <li>Item de lista</li> representa um item específico dentro de uma lista.

Link

(Voltar)

A tag <a> é utilizada para criar hyperlinks, possibilitando que os usuários naveguem para outras páginas ou seções dentro do mesmo documento. Por exemplo, a marcação <a href="https://www.exemplo.com">Visite o Exemplo</a> cria um link que direciona o usuário para o site "Visite o Exemplo" com o texto âncora "Visite o Exemplo".

Linha Horizontal

(Voltar)

A tag <hr> é utilizada para criar uma linha horizontal, que serve geralmente para separar diferentes conteúdos em uma página. Por exemplo, a marcação <hr> insere uma linha horizontal que pode ajudar a organizar visualmente o layout do texto.

Subscrito

(Voltar)

A tag <sub> é utilizada para exibir texto como subscrito, sendo frequentemente aplicada em fórmulas químicas ou notações matemáticas. Por exemplo, a marcação H<sub>2</sub>O H2O representa a fórmula da água, onde o "2" aparece como um subscrito, indicando a quantidade de átomos de hidrogênio na molécula.

Sobrescrito

(Voltar)

A tag <sup> é utilizada para exibir texto como sobrescrito, sendo comumente empregada para indicar potências ou notas de rodapé. Por exemplo, a marcação E = mc<sup>2</sup> representa a famosa equação de Einstein E = mc2, onde o "2" aparece como um sobrescrito, denotando a potência de "c".

Citação em Bloco

(Voltar)

A tag <blockquote> é utilizada para definir uma citação em bloco, sendo comumente aplicada a citações mais longas. Por exemplo, a marcação <blockquote>"Esta é uma citação."</blockquote> apresenta a frase "Esta é uma citação" em um formato destacado, indicando que se trata de uma citação de outra fonte.

Fonte (<font>)** (obsoleta)**

(Voltar)

A tag <font> era utilizada para definir o tipo de fonte, tamanho e cor do texto. No entanto, essa tag está obsoleta em HTML5 e não deve ser utilizada. Em vez disso, recomenda-se o uso de CSS para estilizar o texto. Um exemplo do uso da tag seria <font color="red">Texto colorido.</font>, que renderizaria o texto "Texto colorido" em vermelho.

Texto Deletado (<del>)

(Voltar)

A tag <del> é utilizada para indicar texto que foi deletado ou removido, geralmente sendo exibido com um risco atravessado. Por exemplo, a marcação <;del>Texto deletado.</del> apresenta a frase "Texto deletado" com um traço sobre ela, sinalizando que o conteúdo não é mais relevante ou foi excluído.

Abreviação (<abbr>)

(Voltar)

A tag <abbr> é utilizada para definir uma abreviação ou acrônimo, permitindo a inclusão de uma descrição completa por meio do atributo title. Por exemplo, a marcação <abbr title="Hypertext Markup Language">HTML</abbr> representa a abreviação "HTML" e, ao passar o mouse sobre ela, o usuário pode ver a definição completa "Hypertext Markup Language".
